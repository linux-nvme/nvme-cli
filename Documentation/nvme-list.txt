nvme-list(1)
============

NAME
----
nvme-list - List all recognized NVMe devices on the host machine

SYNOPSIS
--------
[verse]
'nvme list'

DESCRIPTION
-----------
Scan the host's sysfs tree for NVM Express devices. Return the /dev node
for those devices, as well as some pertinant information about them.

OPTIONS
-------
None

EXAMPLES
--------
No examples yet.
