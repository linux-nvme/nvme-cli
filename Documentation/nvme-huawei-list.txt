nvme-list(1)
============

NAME
----
nvme-huawei-list - List all recognized Huawei NVMe devices

SYNOPSIS
--------
[verse]
'nvme huawei list'
			[<global-options>]

DESCRIPTION
-----------
Scan the sysfs tree for NVM Express devices and return the /dev node
for those Huawei devices as well as some pertinent information about them.
